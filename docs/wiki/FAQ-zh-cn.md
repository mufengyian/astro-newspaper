# FAQ

[English](FAQ-en) · [返回 Wiki 首页](Home)

## 这是一个可安装的主题包吗？

不是。当前更适合把它当作 starter 仓库或模板仓库来使用，推荐方式是 clone 或 Use this template。

## 可以只写中文吗？

可以。英文结构可以暂时留空，不必强制同步发布；不过保留双语目录结构通常更利于后续扩展。

## 可以只用 Markdown 吗？

可以。`.md` 和 `.mdx` 都支持，不需要组件能力时完全可以只用 Markdown。

## 为什么首页社交链接默认不显示？

因为默认公开展示占位用户名、占位邮箱并不专业，也不安全。只有你主动填写 `siteConfig.socialLinks` 时才会显示。

## 为什么没有配置 `PUBLIC_SITE_URL` 就看不到完整 RSS / sitemap？

这是主题的安全默认。没有真实公开地址时，宁可不输出这些元信息，也不要输出错误版本。

## 这一版应该从哪里改视觉风格？

优先从 [`src/styles/tokens.css`](../../../src/styles/tokens.css) 开始，而不是先改组件级 CSS。

## 这一版主题默认是浅色还是深色？

默认跟随系统主题；用户点击切换后，偏好会持久化到本地存储。

## 适合直接拿来做文档站吗？

适合轻量文档、笔记或知识库首页。如果你需要完整的多级侧边导航、版本管理和文档专用信息架构，专业文档框架会更合适。
