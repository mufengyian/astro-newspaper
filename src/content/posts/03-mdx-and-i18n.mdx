MDX allows you to use components directly in your Markdown content. This theme includes a few to get you started, but you can create your own.

### Callout Component

Here is an example of a simple callout component you could create and use:

```mdx
import Callout from '~/components/Callout.astro';

<Callout type="info">
  This is an informational message. You can use components to add rich, interactive elements to your posts.
</Callout>
```

### Bilingual Support

This theme is designed from the ground up for bilingual content. To create a translated version of this post, simply create a new file (e.g., in Chinese) and assign it the same `translationKey` in its frontmatter:

**English Post (`mdx-and-i18n.md`):**
```yaml
---
title: Beyond Markdown: MDX and Bilingual Content
locale: en
translationKey: mdx-and-i18n
---
```

**Chinese Post (`mdx-和-i18n.md`):**
```yaml
---
title: 超越 Markdown：MDX 与双语内容
locale: zh-cn
translationKey: mdx-and-i18n
---
```

The theme will automatically handle routing and provide links to switch between the two versions.
