这篇文章主要用来检查主题对常见内容块的处理是否稳定。

## 代码块

```ts
export function sum(a: number, b: number) {
	return a + b;
}
```

代码块右上角有复制按钮，方便直接取命令或片段。

## 引用

> 主题的目标不是增加尽可能多的功能，而是把常见功能放在合适的位置。

## 表格

| 项目 | 说明 |
| --- | --- |
| 代码复制 | 文章页增强 |
| 目录 | 根据标题生成 |
| 阅读进度 | 页面顶部进度条 |
| 上下篇导航 | 文章底部 |

## 列表

- 无序列表适合写要点。
- 有序列表适合写步骤。
- 表格适合做字段或功能对照。

## 图片

封面图通过 `astro:assets` 输出响应式版本，列表页和文章页会使用不同的 `sizes` 配置。
