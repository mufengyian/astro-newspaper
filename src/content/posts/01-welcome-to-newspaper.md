Newspaper is an Astro blog theme built for writers, publishers, and content creators. It combines a clean, readable design with a powerful set of features to help you create a beautiful and performant site.

### Key Features

- **Typography-focused**: A calm, readable design that puts your content first.
- **Performance by Default**: Built with Astro for a fast, lightweight experience with zero client-side JavaScript by default.
- **MDX & Components**: Go beyond standard Markdown. Embed interactive components directly in your posts.
- **Bilingual Content**: Pre-configured for both Chinese and English, with easy-to-manage i18n routing.

This demo site will walk you through the core features of the theme. Feel free to explore the different post types and see what's possible.
