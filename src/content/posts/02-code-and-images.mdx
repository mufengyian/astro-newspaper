This theme provides beautiful styling for code blocks and leverages Astro's built-in `astro:assets` for powerful image optimization.

### Code Blocks

Here is a JavaScript code block with syntax highlighting powered by Shiki:

```javascript
import { defineConfig } from 'astro/config';

export default defineConfig({
  site: 'https://example.com',
  integrations: [],
});
```

You can customize the theme and language support in your `astro.config.mjs` file.

### Image Handling with `astro:assets`

The cover image for this post is a local file processed by Astro. The theme automatically handles responsive sizes and modern formats like WebP and AVIF.

Here's how you can use the `<Image />` component in your MDX files:

```mdx
---
import { Image } from "astro:assets";
import myImage from "../../assets/my-image.png";
---

<Image src={myImage} alt="A description of my image." />
```

This approach ensures your images are always optimized for performance without sacrificing quality.
